What is GIT

GIT is a distributed version-control system for tracking changes in any set of files, originally designed for coordinating work among programmers cooperating on source code during software development. Its goals include speed, data integrity, and support for distributed, non-linear workflows

Why is it used?

GIT is tool which enables it users to work on projects and distribute the same among their co-workers. It saves the history of all the changes occured to the and also the information of who made what changes. This can be used to bring the project back in time if something goes wrong.

Avantages of GIT

  • Distribution of work
  • Incorporate changes
  • Risk-free
  • Open-source
  • Platform Independent

Some GIT commands

Command Description
git clone Copy a git repository
git add Add file to staging area
git commit Saving the changes in staged files
git status Info about changes and commits
git branch Create a new timeline for project
git merge Bring changes from other branches to main project
git pull Bring changes from remote repo to local
git push Save local commits to remote repo
